# 🌳 心灵树洞

一个温暖的 AI 情感陪伴助手。这里是一个安全的空间，你可以倾诉心事、分享日常，我会安静地听，给你温暖的回应。

## ✨ 核心功能

- **💬 温暖对话** - AI 以理解、非评判的方式回应你的倾诉
- **🎭 情绪识别** - 自动识别你当下的情绪状态
- **📝 情绪日记** - 记录每天的心情变化
- **📊 心情周报** - 可视化展示你的情绪趋势

## 🎯 设计理念

**「不解决问题，只陪伴理解」**

心灵树洞不是心理咨询，也不是问题解决器。它是一个愿意倾听的朋友，在你需要的时候给予温暖的陪伴。

## 🚀 快速开始

### 1. 克隆项目

```bash
git clone https://github.com/YOUR_USERNAME/mind-treehole.git
cd mind-treehole
```

### 2. 安装依赖

```bash
npm install
```

### 3. 配置环境变量（可选）

```bash
cp .env.example .env
# 编辑 .env 文件，添加你的 OpenAI API Key
```

> 💡 **提示**：即使不配置 API Key，应用也可以运行。它会使用预设的温暖回复。

### 4. 启动开发服务器

```bash
npm run dev
```

打开 [http://localhost:3000](http://localhost:3000) 查看效果。

## 🛠️ 技术栈

| 类别 | 技术 | 选择理由 |

## 🎨 角色设计

### 小树 - 温柔倾听者

**性格特质：**
- 温暖、善解人意
- 善于倾听，不急于给建议
- 用朋友的方式交流，不是专家姿态

**说话风格：**
- 简短但有温度（通常 2-4 句话）
- 适当使用表情符号（🌸、💛、🫂）
- 共情回应："我能感受到..."、"听起来..."

**原则：**
- ✅ 先理解感受，再回应
- ✅ 接纳所有情绪，不评判
- ✅ 温暖陪伴，让对方不孤单
- ❌ 不说教、不给非请求的建议
- ❌ 不说"别难过"、"想开点"

## 📊 情绪识别

当前使用关键词匹配的方式识别情绪，支持：

| 情绪 | Emoji | 示例关键词 |
|------|-------|-----------|
| 开心 | 😊 | 开心、高兴、快乐、笑 |
| 难过 | 😢 | 难过、伤心、哭、痛苦 |
| 焦虑 | 😰 | 焦虑、担心、紧张、压力 |
| 愤怒 | 😤 | 生气、愤怒、烦、讨厌 |
| 孤独 | 🥺 | 孤独、孤单、一个人 |
| 平静 | 😌 | 平静、还好、一般 |

> 🔮 **未来优化方向**：可以接入 AI 做更精准的情绪分析，或使用专门的 emotion detection 模型。

## 🔒 隐私说明

- 所有对话数据存储在用户浏览器的 localStorage 中
- 不会向任何第三方服务器发送数据（除了可选的 OpenAI API）
- 用户可以随时清除浏览器数据来删除所有记录
